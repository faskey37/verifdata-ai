## 🚀 Features

### Core Features
- **Excel/CSV Upload** - Upload .xlsx or .csv files with preview
- **Phone Number Validation** - Format validation, country detection, carrier identification
- **Duplicate Detection** - Find duplicates in phone numbers, names, and addresses
- **Data Quality Checks** - Missing fields, invalid emails, suspicious names
- **Verification Status Dashboard** - Track customer verification status
- **Customer Reverification** - Send SMS OTP, email verification, generate links
- **Risk Scoring** - Auto-calculate risk scores (Trusted/Review/High Risk)

### Advanced Features
- **Interactive Analytics** - Real-time charts and metrics
- **Bulk Actions** - Mass verify, delete duplicates, export data
- **AI-Assisted Data Cleaning** - Auto-capitalization, address standardization, fake name detection
